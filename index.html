    The Web
    The World Wide Web is the largest computer network. Is mostly written in HTML. Invented in the 1990s and it has around 30 million pages. When a person goes to a web page the computer sends a HTTP Request to a server. The server finds the correct HTML document and sends it back to the user's computer. A web browser interprets the page and displays it for the user.
    HTML
    HTML stands for Hypertext Markup Language. HTML documents contain text content which is "what you see",markup which styles "how it looks", reference to other documents like images or videos,as well links to other pages. A web page is made out of tags, elements, and attributes.

    Grammar

    Independent and Dependent Clauses.

    Dependent clause is a group of words that also contains a subject and a verb, but it is not a complete thought. Because it is not a complete thought, a dependent clause cannot stand on its own as a sentence; it is dependent on being attached to an independent clause to form a sentence. • Dependent clauses can often be identified by words called dependent markers, which are usually subordinating conjunctions. If a clause begins with one of these words, it is dependent and needs to be attached to an independent clause. • Common dependent markers: after, as, although, because, before, even though, if, once, rather than, since, that, though, unless, until, when, whenever, whereas, while, among others.
